# AzureArcTools
This repository gathers a set of tools to work with your Azure Arc Environment


## Table of Contents

- [AgentUpdates](/AgentUpdates/) - ARM templates to update Azure Arc agents on Windows servers, including proxy support
- [AMAProxy](/AMAProxy/) - Azure Policy definitions and scripts to configure proxy settings for Azure Monitor Agent (AMA) on Azure Arc servers
- [ArcEnableServerManagement](/ArcEnableServerManagement/) - Scripts and queries to enable Azure Arc server management features
- [ArcSQLServersBPA](/ArcSQLServersBPA/) - Tools to enable and configure SQL Server Best Practices Assessment for Azure Arc-enabled SQL Servers
- [ArcSQLSetLicenses](/ArcSQLSetLicenses/) - Scripts to manage and reclassify SQL Server licenses on Azure Arc-enabled servers
- [ArSQLMaintenance](/ArSQLMaintenance/) - Maintenance scripts to remove orphaned Azure Arc SQL Server resources
- [CleanupAgent](/CleanupAgent/) - Scripts to clean up and remove Azure Arc agent installations
- [ESUSetLicenses](/ESUSetLicenses/) - Tools to manage Extended Security Update (ESU) licenses for Azure Arc servers
- [Extensions](/Extensions/) - Scripts to install, update, remove, and manage Azure Arc extensions
- [MigrateMMAtoAMA](/MigrateMMAtoAMA/) - Migration scripts to transition from Microsoft Monitoring Agent (MMA) to Azure Monitor Agent (AMA)
- [Workbooks](/Workbooks/) - Azure Monitor workbook templates for Azure Arc resources, including Change Tracking
